学习配置环境和初始化项目
===

Install
-------
[安装依赖]

```bash
$ npm install -g yarn react-native-cli
```

Init
-------
[初始化]

```bash
$ react-native init 项目名称
```

Reload from git clone
-------
[git clone 下来的项目重新安装依赖]

```bash
$ yarn install
```
